Request all countries

https://country-api-production.up.railway.app/countries

Request by country

  • Country name has to be uppercase.
  • For countries with spaces use Percent Encoding (ex. United%20States)

The following code will return the country Mexico.

https://country-api-production.up.railway.app/countries/name/Mexico
